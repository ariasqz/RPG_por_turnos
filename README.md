# RPG por Turnos

RPG por turnos en el navegador desarrollado con JavaScript vanilla. Proyecto creado para practicar Programacion Orientada a Objetos.


## Caracteristicas

- Sistema de combate por turnos contra 4 enemigos diferentes
- Tres acciones disponibles: atacar, lanzar hechizo y curarse
- Regeneracion de HP al final de cada turno
- Sistema de niveles y experiencia
- Barras de HP y mana con animaciones
- Indicadores de dano flotantes

## Tecnologias

- HTML
- CSS
- JavaScript (ES6+)
